Cloud OJ 是一个微服务架构的 Online Judge 系统，基于 Spring Cloud、Vue.js。

- 支持 Docker 部署
- 代码高亮
- 可扩展判题服务

## Supported Languages

- C
- C++
- Java
- Python
- Bash Shell
- C#
- JavaScript
- Kotlin
- Go

## Build & Debug

需要以下环境:

- CMake 3.16+
- Linux / WSL
- Maven 3.8
- OpenJDK 17+
- Node.js v16
