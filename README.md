Cloud OJ 是一个微服务架构的 Online Judge 系统，基于 Spring Cloud、Vue.js。

- 支持 Docker 部署
- 支持 SQL 判题（SQLite）
- 代码高亮
- 可扩展判题服务

## Supported languages

- SQLite
- C
- C++
- Java
- Python
- Bash Shell
- C#
- JavaScript
- Kotlin
- Go

## Build & Debug

需要以下组件:

- CMake 3.16
- Linux / WSL
- Maven 3.8
- OpenJDK 11+
- Node.js v16
